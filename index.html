A. S. I. R

Amateur Satellite Image Reception

My hobby is repelling pictures from weather satellites

Receiving such images is not the easiest thing to do

I decided to write a program that would facilitate this task

The main goals of the program are:

  • determining the current position of the satellites
  • prediction of the next crossings over the horizon
  • drawing a map with the flight trajectory
  • communicating with sdr in order to receive iq samples
  • signal demodulation and conversion
  • automatic placement of decoded images in appropriate places with appropriate tags
  • 100% offline so it can be used anywhere in the world!
