Receiving images

Acquiring satellite images is a long and complicated process

Steps to receive LRPT image from METEOR-M N2:

  • find the next pass of the satellite over the horizon
  • run the satellite tracking program and sdr
  • capture raw IQ data from sdr
  • demodulate data using QPSK demodulation
  • decode data from raw binary file
  • combine 3 gray bw frames into colored rgb image
  • apply color correction to blue channel and stretch image
